## 🎨 职业路径算法

### 技术专家
- 技能评估: Q1 (技术问题) + Q2 (学习新技术)
- 适合：深度技术钻研、持续学习

### 技术管理  
- 领导力: Q3 (沟通) + Q4 (领导) + Q5 (规划)
- 适合：团队管理、技术决策

### 产品技术
- 综合能力: Q3 (沟通) + Q7 (客户交流)
- 适合：技术与商业结合

### 独立创业
- 创业精神: Q9 (创业兴趣) + Q2 (学习) + Q12 (收入)
- 适合：自驱力强、风险承受

### 学术研究
- 研究倾向: Q31 (论文/文献) + Q32 (严谨验证) + Q33 (研究表达)
- 适合：长期研究、学术积累

### 教育培训
- 教学能力: Q8 (培训指导) * 2 + Q3 (沟通)
- 适合：知识分享、培养人才

## 🚀 部署步骤

### 快速部署（推荐）
```bash
cd career-test-cn
./deploy.sh
```

### 手动部署
```bash
# 1. 安装依赖
npm install

# 2. 本地测试
npm run dev

# 3. 构建项目
npm run build

# 4. 部署到 Vercel
vercel --prod
```
