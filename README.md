# 🎯 职业发展测评

一个专业的职业发展测评工具，帮助用户找到最适合的职业发展路径。

## ✨ 特性

- 📊 15 道精心设计的测评问题
- 🎯 5 种职业发展路径推荐
- 💼 详细的职位建议和技能要求
- 📱 完全响应式设计，支持移动端
- 🚀 基于 Next.js 14，性能优异

## 🎨 职业路径

1. **技术专家** - 深入钻研技术，成为领域专家
2. **技术管理** - 带领技术团队，平衡技术与管理
3. **产品技术** - 连接技术与业务，打造优秀产品
4. **独立创业** - 创办公司或成为独立开发者
5. **教育培训** - 分享知识，培养技术人才

## 🚀 部署到 Vercel

### 方法 1: 通过 Vercel CLI（推荐）

1. 安装 Vercel CLI：
```bash
npm i -g vercel
```

2. 在项目目录中运行：
```bash
cd career-test-cn
vercel
```

3. 按照提示完成部署：
   - 登录你的 Vercel 账号
   - 选择项目设置
   - 等待部署完成

### 方法 2: 通过 GitHub + Vercel Dashboard

1. 将代码推送到 GitHub：
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```

2. 访问 [vercel.com](https://vercel.com)
3. 点击 "New Project"
4. 导入你的 GitHub 仓库
5. Vercel 会自动检测 Next.js 项目并配置
6. 点击 "Deploy" 完成部署

### 方法 3: 一键部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=YOUR_GITHUB_REPO_URL)

## 💻 本地开发

1. 安装依赖：
```bash
npm install
```

2. 运行开发服务器：
```bash
npm run dev
```

3. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)
